Keyboard Navigation Guide

This guide explains how to navigate the KitchenSink application using only your keyboard.

Basic Keyboard Navigation

You can navigate through the KitchenSink application using these basic keyboard controls:

  • Tab: Move forward to the next focusable element
  • Shift + Tab: Move backward to the previous focusable element
  • Enter: Activate buttons, links, or submit forms
  • Space: Activate buttons or toggle checkboxes
  • Arrow keys: Navigate within some components like tables

Note: The tab order follows a logical sequence through the page, moving from top to bottom and left to right.

Keyboard Shortcuts

The following keyboard shortcuts are available in the KitchenSink application:

Shortcut Function Description
Alt + R Focus Register Button Quickly jump to the Register button in the member registration form
Alt + 1 Skip to Main Content Same function as the skip link, jumps directly to the main content area

Note: On macOS, use ⌥ Option instead of Alt

Skip to Content Link

At the beginning of each page, there is a "Skip to main content" link that allows you to bypass the navigation and header sections:

Skip to main content

This link is initially hidden but becomes visible when you:

  1. First load the page and press Tab
  2. Or press Tab when focus is at the top of the page

Pressing Enter when this link has focus will take you directly to the main content area.

Focus Indicators

To help you see which element currently has keyboard focus, all interactive elements display a distinctive blue outline when focused:

This is how a focused element appears

This visual indicator helps you track your position as you navigate through the application with the keyboard.

The focus indicators have been designed to:

  • Have high contrast (meeting WCAG 2.1 requirements)
  • Be clearly visible against all background colors
  • Be distinct from hover states

Form Navigation

The member registration form can be navigated as follows:

  1. Use Tab to move through form fields in order
  2. Complete each field with keyboard input
  3. When the Register button has focus, press Enter or Space to submit the form
  4. Form validation messages will be announced by screen readers

Remember that you can use Alt + R to jump directly to the Register button from anywhere in the application.

Table Navigation

The Members table can be navigated using the keyboard:

  1. Tab to the table (it has a tabindex of 0)
  2. Once the table has focus, use arrow keys to navigate between cells:
    • ↑ and ↓: Move between rows
    • ← and →: Move between columns
  3. When a cell containing a link has focus, press Enter to activate the link

This keyboard navigation guide was created to improve accessibility in accordance with WCAG 2.1 guidelines.
